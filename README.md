# Apple-Picker-
A high-speed arcade game developed as part of a Unity learning path. The goal is to catch falling apples using a stack of baskets.

## Key Features
* **Object Instantiation:** Dynamic spawning of apple trees and apples.
* **Game Logic:** Increasing difficulty and basket destruction mechanics.
* **UI Integration:** Real-time score tracking and "High Score" persistence.
* **Input Management:** Smooth mouse-based basket movement.

## Tech Stack
* **Engine:** Unity 2022.3+ (or Unity 6)
* **Language:** C#
* **Assets:** Custom simple 3D primitives and materials.

## How to Play
Move your mouse horizontally to control the baskets. Catch the green apples! If an apple hits the ground, you lose one basket.
